# Lawrence Rhee

Personal website and research journal, built for DigHum 150A at UC Berkeley.
Plain HTML and CSS, no framework, no build step.

## Structure

```
index.html           Home page
about.html            About me
project.html          Final project page
journal/              Research journal (blog posts)
assets/styles.css     Site stylesheet
```

## Running locally

```
python -m http.server 8080
```

Then open <http://localhost:8080>.
